# API for Platos by Gloria

## Routes
---

### Plates
- **Get** all plates:
> /api/plates

- **Get** one plate:
> /api/plates/```:id```

- **Post** one plate:
> /api/plates

- **Patch** one plate:
> /api/plates/```:id```

- **Delete** one plate:
> /api/plates/```:id```

- **Delete** one or multiple plates:
> /api/plates
>> requires a body ```[arr of ids]```
